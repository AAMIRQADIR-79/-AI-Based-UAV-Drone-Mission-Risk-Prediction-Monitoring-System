# AI-Based-UAV-Drone-Mission-Risk-Prediction-Monitoring-System
 An intelligent UAV (drone) mission planning system that uses Machine Learning, real-time weather data, and simulation to predict mission risk and assist operators in making safer decisions before and during flight.

Project Status: Work in Progress

This project is currently under active development as part of a Final Year Engineering Project.

🔄 Current Progress
✅ Literature survey completed
✅ Problem statement and objectives defined
✅ Dataset generation and preprocessing
✅ Machine Learning model (Random Forest) implemented
✅ Basic Streamlit dashboard developed
✅ Integration of real-time weather API
🔄 Dynamic route simulation (in progress)
🔄 Advanced map visualization and drone animation (in progress)
🔄 Model optimization and Explainable AI improvements
